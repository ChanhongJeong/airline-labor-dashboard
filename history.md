**구조 (양호)** - `getCountryForYear(c, year)` 함수가 `yearOverrides[year]`에서 `labor / wagePremiums / fixedTermContract / annualLeave / workRules / hiringPlaybook / thirteenthMonth / detailedLabor / contractRequirements` 9개 필드를 시점별로 끌어오는 구조가 잘 구현됨. year selector로 과거 연도 선택 시 해당 연도 시점 노동법이 반영됨.

**데이터 (편차 있음)** - 전체 16개국 중 `yearOverrides`가 채워진 국가는 8개에 불과:
| 구분 | 국가 |

| ✅ yearOverrides 보유 (8개) | 한국, 대만, 홍콩, 일본, 베트남, 태국, 말레이시아, 인도네시아 |
| ⚠️ yearOverrides 미보유 (8개) | **중국, 러시아, 몽골, 마카오, 사이판, 필리핀, 라오스, 싱가포르** |

**후속 권장 작업** - 우선순위 높은 미보유 국가(중국, 필리핀, 싱가포르 등)에 대해 `yearOverrides` 추가 입력. 예시:
- 중국: 2024.5 최고인민법원 무고정기한 전환 전형판례 / 2025.7 플랫폼 노동자 직업병 보험 확대 / 2023.1 부녀자권익보장법 성희롱 8가지 의무 → 시점별 분기 가능
